# Cruise Hackathon Project

A smart cruise booking platform that allows users to:
- Browse cruises
- Book cabins
- Reserve restaurants, shows, and casino activities
- Prevent schedule conflicts
- Get activity recommendations

## Tech Stack
- Python
- Flask
- SQLite
- HTML / CSS / JavaScript

## Features
- Cruise search
- Cabin booking
- Activity scheduling
- Conflict detection
- Dashboard analytics

## Run Locally
pip install -r requirements.txt
python app.py
